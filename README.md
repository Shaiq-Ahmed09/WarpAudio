# AuraSpace-Music-Player
simple music player app. just upload your songs here and you are good to go. totally customizable by you!
"Run python server.py to start the AuraStream backend, then open http://127.0.0.1:8000 to upload and stream your local music files with full persistent storage."

this app uses the youtube's api key to access the youtubes massive library but here is a catch. You cannot access the key i have implemented so you have to request a key from youtube ("You can get it for free") then implement it and access it.
